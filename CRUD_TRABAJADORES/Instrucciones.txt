Para desplegar este sistema se debe seguir los siguientes pasos:
1) Clonar o descargar el proyecto del GITHUB
2) Mover al proyecto a la carpeta de apache y redireccionar hasta la carpeta public que es donde se encuentra el index
3) Modificar el hosting, usuario y contraseña del archivo que se encuentra en la carpeta config llamado Database.php
con las credenciales de su base de datos.
4) En el gestor de bases de datos abrir el archivo SQL que se encuentra en la carpeta BD SQL y copilarlo.

Con eso esta todo listo gracias por su atención.....
